# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-06
### Added
- Streamlit interactive web application (`app/main.py`) serving 6 distinct ML modules.
- Unified project structure standardizing `notebooks/`, `models/`, and `data/` directories.
- Comprehensive portfolio documentation including architecture diagrams, data flow diagrams, and recruiter guides.
- Streaming `SGDClassifier` pipeline for simulated real-time fraud detection.
- `CuPy` based Funk SVD recommendation engine.

### Changed
- Consolidated `project1_ecommerce_intelligence`, `project2_financial_fraud_detection`, and `project3_fashion_image_classifier` into a monolithic platform architecture.
